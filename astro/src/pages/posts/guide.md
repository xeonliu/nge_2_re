# 介绍

## 01. 序言

《新世纪福音战士：被创造的世界 - another cases -》于 2006 年 4 月 27 日由万代发售。这是一款为 PSP 平台制作的游戏。游戏在移植了 2004 年发售于 PS2 平台的《新世纪福音战士 2》的基础上，还新增了许多附加内容并增强了游戏体验。万代还发行了一款限定版本，它采用了一个标题为“Memorial Box”的特殊包装。包装盒和光盘上使用了与平装版不同的艺术图像。更重要的是，该版本还包含了两份 UMD 视频，即《死与新生》和《真心为你》两部电影。

这部游戏的类型（通常归类为“世界模拟”）很难描述，鉴于它混杂了日常模拟的元素，类似电子宠物的游戏进程以及实时策略对战，这些都和 EVA 系列的剧情以及其他片段混杂在一起。在游戏的每个场景中，你会扮演新世纪福音战士角色中的一位并且体验它的日常生活。不管怎么说，倘使这不是 EVA 改编的游戏中最好的，那也至少是其中之一。EVA 系列的粉丝一定会享受这款游戏。

这个教程并不仅仅是一个通关教程。由于这款游戏的开放世界特性，写一个通关教程是毫无意义的。本文更像是一个能够提升游戏体验的问答/指南，包含了菜单选项和可交互物品的介绍，以及对战策略和各种提示与建议。本文面向不懂（或不完全理解）日语的洋人，而日语又是这款游戏唯一支持的语言。原谅我任何在顺序组织上的无序；这款游戏如此复杂、拥有如此多的选项以至于在撰写本文时很难找到合适的顺序。

尽管并被必要，我建议你学下片假名，鉴于这可能帮助你浏览游戏菜单：很多菜单都是按单词表顺序来的，角色名称也是一样。

## 02. PS2/PSP 差异

这款游戏最初于 2004 年在 PS2 平台发布。在 2006 年 4 月 27 日，它被移植到 PSP，对游戏进行了强化并引入了新的游戏元素，增添了游戏的维度。这篇教程为 PSP 版本撰写，机关游玩 PS2 版本的人也可能会找到有用的信息。

下面是一些版本差异：

PSP 版添加了新的剧情(Scenarios)，而且大多数原有的剧情也被修改了（允许你游玩更长的故事）。

引入了“Impulse 参数”。它与 A.T. 一样重要，决定特定时刻你能做的事情。完成行动需要付出一定量的“Impulse”。

引入了“成功/失败 Pulse”，从此技能不再是随机增加的。

# 如何游玩

## 03. 游戏流程

《新世纪福音战士 2：被创造的世界》的游戏流程分为以下四个部分：自由回合、对战、场景和脉冲。在这一节我们将讲述这些部分的内容和游玩方法。

### a) 自由回合

这是游戏的主要环节。在这个环节中你可以与其他人物交互，做出各种行为（吃饭、喝水、睡觉、学习、如厕、沐浴……），访问不同地点，进行骇入，制作卡片、训练技能等等。部分角色在其剧情期间可以执行特殊的行为。

在自由回合中，屏幕四周的菜单大致长这个样子：

```
--------------------------------------------------
|时钟 / 防御状态                      其他角色的状态 |
|                                                |
|                                                |
|                                                |
|                                                |
|                                                |
|                                                |
|                                                |
|                                                |
|                                      (可用指令) |
|                                                |
|你的 A.T. ..................................... |
| ................................. 你的 IMPULSE |
--------------------------------------------------
```

“时钟”显示游戏中的时间。“防御状态（DefCon）”标识使徒下次入侵的大致时间。“可用指令”是一组文本框，当你靠近可交互物品时会显示出来。A.T.和 IMPULSE 栏对于游戏进程分厂重要（参见“角色参数”一节）。在右上角，你会看到在场的角色的名字、A.T.以及 IMPULSE。

### b) 战斗回合

在自由回合中，当所有 DefCon（防御状态，即时钟旁边的蓝色椭球）中所有椭球全部变红后，一切行动将终止，警报声响起，战斗回合将开始。

只有 EVA 驾驶员和美里的场景才会完整地经历这一阶段。每当使徒（Angel）攻击时，其他角色的场景会显示由美里讲述的简短简报，包含战斗的静态图片以及战斗结果的信息。

而 EVA 驾驶员则必须击败敌人。在一连串介绍对入侵的介绍之后，战斗将在一个二维地图上展开，地图上有不同的图标代表 EVA、敌人以及可能出现在战场上的后勤设施。你将控制 EVA 的图标，执行各种命令。战斗命令菜单会根据你的位置和周围的元素显示不同的选项。只有当敌人在你的武器或近战攻击范围内时，才能攻击敌人。当你攻击或被攻击时，将显示一段壮观的战斗 3D 动画。

屏幕的上方会显示敌人的生命值条，而下方会显示 EVA 的生命值条和驾驶员的同步率。你的主要目标是在敌人到达地下都市（Geofront）或 NERV 总部之前，将敌人的生命值条减至零。如果你的 EVA 被摧毁，游戏将结束。

在美里的场景中，你无法直接控制 EVA 单位。你将选择它们的装备和初始位置，并在战斗开始后给驾驶员下达命令。目标仍然是将敌人的生命值条减至零。

战斗回合结束后，你会收到结果报告，然后可以保存游戏。新的一章将开始。你的角色的冲动条将恢复，A.T.值也可能有所变化。

有些特殊战斗采用不同的方法，有些甚至可能在非驾驶员和非美里的场景中也具备互动性。有关这一阶段的进一步解释以及战胜敌人的策略，请参阅“战斗”一节。

### c) 场景

场景是一组静态图片，配有音乐、旁白和文字来展开故事情节。它们可能在自由回合中当你进入新地点时出现，也可能在睡眠期间出现。除此以外也可能在战斗中途出现。

场景分为两种类型：关键场景和随机场景。

关键场景讲述每个场景的主要故事，并且是独一无二的。要观看最好的结局，必须触发这些关键场景。要触发关键场景，你需要满足某些条件，例如拥有某个物品、遇见某人、访问特定地点、达到最低的 A.T.值或技能等。

另一方面，随机场景可以在任何场景中发生，但它们不一定会被触发。它们是来自剧集中的简短场景，也可能是一些其他事件，但它们不属于当前剧情的主要情节。根据你的游戏方式、角色的属性和许多其他因素，一些场景可能会被触发（但总是有随机因素）。获得特殊武器或遭遇某些使徒可能需要特定场景在此前被触发。随机场景很棒，因为它们为游戏提供了很多变化。

每场战斗前（有时也在战斗后）都会播放一个介绍场景。

请注意，真嗣的第一个场景只有关键场景。

### d) 脉冲

这是 PSP 版本游戏的独有内容。严格来说，它并不是一个游戏回合，而是在自由回合阶段中的一个互动事件。为了提升技能，或者执行其他一些操作（如黑客攻击），你将被传送到一个新屏幕，在那里你需要快速地按下正确的 10 个按钮序列来通过测试。如果你按错按钮，脉冲将结束。

脉冲有两种类型。你可以在脉冲屏幕的右上角查看脉冲的类型。

#### 效率评判

你将根据成功输入的数量获得奖励。输入的数量越多，奖励就越高。

在提升技能时，成长幅度通常从+0（没有成功输入）到+6（10 次成功输入）不等。如果失败，它可能只会对你的 A.T.值和当前情绪造成负面影响。满分 10 次输入将显示为“Perfect”。

#### 成功或失败

只有在成功输入 10 个按钮时，才能获得奖励。失败这个脉冲甚至可能导致游戏结束（例如在 “熔岩潜行者” 事件中），同时也可能对你的 A.T.值和当前情绪产生负面影响。满分 10 次输入将显示为“Success”。此类型的脉冲在其他部分将简写为“S/F PULSE”。

在脉冲屏幕的中央，会有一个绿色区域。一系列随机生成的三条横向彩虹色条将在其中穿过。这些条分别代表三角形键、O 键和 X 键。你需要在对应的条充满并且其边缘位于绿色区域内时按下相应的按钮（三角形、O 或 X）。如果你按得太快或太慢，你将失败，脉冲将结束。这是一个相当困难的过程，因为这些条充满的速度是随机的，所以没有实际的规律可循。另外，注意有时会有色条在到达绿色区域前就到收回了。不要按任何按钮，否则你的脉冲将失败：这只是一个干扰。一旦脉冲结束，你将得到结果报告，接着自由回合阶段将继续。

绿色区域的宽度和条的速度（换句话说，脉冲的难度）将基于你当前的 A.T.值（无一例外）以及其他技能或参数（根据你所进行的脉冲类型而有所不同）。例如：美里的作业脉冲基于她的工作技能，Pen Pen 的领取优惠券脉冲基于他当前的 WC 值。如果脉冲不基于某个特定技能或参数，它将受到你当前情绪的影响。

一旦你将某个技能提升到其最大值（99），对应的“提升技能脉冲”将从可用命令中消失。

## 04. 菜单

这一章将涵盖游戏过程中最重要的菜单，包括他们的选项和功能。请记住在日本游戏中，O键代表确认，X键代表取消。

### 开始菜单

+ 关卡选择（Scenario Selection）
+ 存档读取（Data Load）
+ 游戏设置（Options）
+ 观察模式（Watch Mode）

下面是对各个选项的解释：

**关卡选择**：允许你通过选择角色关卡开始新游戏。一共可解锁18个关卡，角色可由左侧人物肖像及其名称辨认。通关的关卡（打出好结局）会以微笑肖像和蓝色文字框标识。

**存档读取**：加载存档，最多18个不同存档。

**游戏设置**：修改游戏参数

**观察模式**：该模式下无法控制角色，只能观看AI游玩并与其他角色交互。当完成一个关卡（即使是坏结局）后出现。尽管这模式很没用，你还是可以在其中观赏到一些主游戏中不出现的使徒和情景。我没测过该模式有无时间限制或能否通关。

#### 系统菜单
如名称所示，这个菜单通过在游玩中按下三角形激活。它甚至可以在地图菜单中激活。注意该菜单中并非每个选项都总能使用。

+ 状态
+ 人际关系
+ 机密信息
+ 教程
+ 返回标题
+ 存档
